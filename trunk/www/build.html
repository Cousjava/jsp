JSP Build Instructions

Required Tools

  • J2SE 5.0 or JDK 6
  • Maven 2
  • subversion

Build Steps

  1. Checkout the sources.
    2. % svn co https://svn.java.net/svn/jsp~svn/trunk jspw

  2. Build and install api jar.
    3. % cd jspw/api
    % mvn install

  3. Build api javadoc.
    4. % cd jspw/api
    % mvn javadoc:javadoc

  4. Build and install impl jar.
    5. % cd jspw/impl
    % mvn install
