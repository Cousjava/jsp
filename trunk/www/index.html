JavaServer(TM) Pages Implementation

This project provides a container independent implementation of JSP 2.1.  The main goals are

  • Improves current implementation: bug fixes and performance improvements
  • Provides API for use by other tools, such as Netbeans
  • Provides a sandbox for new JSP features; provides a reference implementation of next JSP spec.

Get Started
How to start
Download
Download the latest JSP implementation in a jar file. It can run with a JavaEE 5 container.
Get Involved
Help improve JSP implementation by providing beedbacks and filing bugs. Those interested in code contribution, can set up a workspace for building and testing.
