JSP Build Instructions

Required Tools

  • J2SE 5.0 or JDK 6
  • Maven 2
  • subversion

Build Steps

  1. Checkout the sources.
    2. % svn co https://jsp.dev.java.net/svn/jsp/trunk jspw

  2. Build a jar.
    3. % cd jspw
    % mvn package

  3. Build javadoc.
    4. % cd jspw
    % mvn javadoc:javadoc
