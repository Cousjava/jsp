Different releases of glassfish contain different snapshots of the JSP implementations.  To use the version of JSP implementation from this project, first download a version of the JSP implementation.
  • Snapshot release
  • version 1.0

To run the downloaded JSP implementation with a web container, dropping it in WEB-INF/lib may not work.   For glassfish 9.1, portions of the lib/appserv-rt.jar has to be replaced with our jar file.

The jar files are also available from the maven repository http://download.java.net/maven/2
